## 🚀 How to Get Started Run Local Development Environment

> ✅ `.env` is ready — ✅`Secret Key` is built in source, no setup needed.

```bash
# 1. Clone project from GitHub
git clone <repo-url>

# 2. Run start script
./get_started.sh
```

📝 The script will start **3 Dockers** : 🐳PostgreSQL · 🐳pgAdmin · 🐳Go server.

They will run automaticaly and test themself, you just only wait and Play!
